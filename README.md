## Enterprise-Grade Crypto Payments & Card Management Infrastructure

**Chainbridge Crypto Card** is a Web3 fintech platform designed to bridge cryptocurrency and traditional payment networks, enabling users to seamlessly spend digital assets in everyday transactions through a secure crypto-backed card ecosystem.

Built with a modern full-stack architecture, the platform combines digital wallet management, payment processing, transaction monitoring, and crypto asset integration to deliver a compliant and user-friendly financial experience.

> **Revolutionizing Crypto Payments**
>
> Empowering users to convert, manage, and spend digital assets through a unified platform that combines crypto wallets, payment cards, transaction tracking, and real-world spending capabilities. The system provides secure financial infrastructure while maintaining the flexibility and transparency of blockchain technology.

## Quick Start

### Clone Repository

```bash
git clone <repository-url>
cd Chainbridge-crypto-card
```

### Install Backend Dependencies

```bash
npm install
```

### Install Frontend Dependencies

```bash
cd client
npm install
cd ..
```

### Configure Environment Variables

```bash
cp .env.example .env
```

Update the required configuration values:

```env
DATABASE_URL=
JWT_SECRET=
API_PORT=
```

### Start Development Environment

```bash
npm start
```

Application will be available at:

```text
Frontend: http://localhost:3000
Backend: http://localhost:5000
```
